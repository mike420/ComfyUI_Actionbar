# ComfyUI Action Bar

<p>A simple actionbar for ComfyUI.</p>

<img width="700" alt="actionbar" src="https://raw.githubusercontent.com/mike420/ComfyUI_Actionbar/refs/heads/main/img/bar.png">

## Installation

### Comfy CLI (recommended)

```bash
comfy node install ComfyUI_Actionbar
```

You can also find it on the [ComfyUI Registry](https://registry.comfy.org/publishers/mike420/nodes/ComfyUI_Actionbar).

### ComfyUI Manager

Search for `ComfyUI_Actionbar` in the Custom Nodes section and install it from there.

### Manual

```bash
cd ComfyUI/custom_nodes
git clone https://github.com/mike420/ComfyUI_Actionbar.git
```

Restart ComfyUI and refresh your browser. That's it.
